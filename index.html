在methods下面定义一个方法：

methods: {
    initMap() {
        //初始化地图
        map = new jsMap(
            Number(this.currentFloor.order),
            Number(this.currentFloor.yaxis),
            Number(this.currentFloor.angle),
            new Config(),
            function () {
                eval(map.options.initFinish)
            }
        )
    },
}

传入要显示的楼层order（默认从0开始） 当前楼层设备点位（yaxis） 当前设备角度（angle）。调用initMap方法 即可显示地图，务必保证在地图dom容器显示之后再调用该方法。

mounted() {
